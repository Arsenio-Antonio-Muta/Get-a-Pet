## Get-a-Pet
This is a dog adoption system I made with NodeJS(Back end) and ReactJS(Front end) where people can apply for dog adoption...


## Technology

Here are the technologies used in this project

*NodeJS
*ReactJS
*MongoDB

## Services used

*Github
